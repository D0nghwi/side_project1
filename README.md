# side_project1 - StudyNote

StudyNote는 노트와 플래시카드를 중심으로 학습을 돕는 AI 학습 보조 웹 애플리케이션입니다.  
사용자가 정리한 노트를 기반으로 학습을 이어갈 수 있도록 
노트 관리, 플래시카드 생성, 노트별 AI 채팅 기능을 제공합니다.

---

## 주요 기능

- **노트 관리**: 생성 / 수정 / 삭제, 태그 기반 정리
- **플래시카드**: 노트 기반 자동 생성, 덱 단위 학습
- **AI 채팅**: 노트별 독립 채팅, 현재 노트 문맥 기반 응답

---

## 기술 스택

**Frontend**
- React, Tailwind CSS

**Backend**
- FastAPI, SQLAlchemy, Pydantic, Local LLM(HyperCLOVAX-SEED 1.5B)

---

## 프로젝트 목표

정리 → 복습 → 질문 으로 이어지는 학습 흐름을  
노트 안에서 완성하는 것을 목표로 합니다.
